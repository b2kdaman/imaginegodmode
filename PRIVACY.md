# Privacy Policy - ImagineGodMode Chrome Extension

**Last Updated:** 2025-01-20
**Version:** 2.3.0

## Overview

ImagineGodMode is a Chrome extension that enhances your experience with Grok's media generation platform. This privacy policy explains what data we collect and how we use it.

## Data Collection

### Analytics

ImagineGodMode uses **Google Analytics 4 (GA4)** to collect anonymous usage statistics. Analytics are **always enabled** and cannot be disabled.

**What we collect:**
- **Anonymous Client ID**: A randomly generated UUID stored locally to identify unique installations
- **Session tracking**: Session duration and engagement metrics (30-minute timeout)
- **Feature usage**: Which features you use (prompts, packs, search, settings, media operations)
- **Event data**: Actions like creating/editing/deleting prompts, pack management, media downloads
- **Settings changes**: Theme, size, language preferences

**What we DO NOT collect:**
- Personal information (name, email, etc.)
- Prompt content or text
- Pack names
- Media URLs or downloaded files
- Browsing history outside of Grok
- IP addresses (anonymized by GA4)
- Any sensitive or identifying information

### Local Storage

The extension stores the following data **locally on your device only**:

**Chrome Storage (chrome.storage.local):**
- Prompt packs and their contents
- Current pack and prompt index
- Prompt ratings

**Browser LocalStorage:**
- Analytics client ID (UUID)
- Analytics session ID and timestamp
- Theme preference
- UI size preference
- Language preference
- Auto-download toggle setting

**This data never leaves your device except for the anonymous analytics events described above.**

## Data Usage

### Analytics Purpose

We collect analytics to:
- Understand which features are most useful
- Identify bugs and errors
- Improve the user experience
- Guide development priorities
- Monitor extension performance

### Data Sharing

- **Analytics data** is sent to Google Analytics 4 and subject to [Google's Privacy Policy](https://policies.google.com/privacy)
- We **do not sell** or share your data with any third parties
- We **do not track** you across other websites
- All analytics data is **anonymous** and **aggregated**

## Security

- **Client ID**: Generated using UUID v4 (cryptographically random)
- **Transmission**: All analytics data sent via HTTPS
- **Storage**: Local data stored using Chrome's secure storage APIs
- **No authentication**: Extension does not require login or account creation

## Your Choices

### What You Can Control

- ✅ Theme, size, and language preferences
- ✅ Auto-download settings
- ✅ All prompt and pack data (export, import, delete)

### What You Cannot Disable

- ❌ Analytics tracking (always enabled)

**Note:** If you do not agree with analytics collection, please do not install or use this extension.
